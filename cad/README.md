# CAD Files

Source files for the kendama geometry. Mesh files (STL, 3MF) are not
committed to the repo — generate them locally or download from GitHub Releases.

**Status:** Not started.

## Target Dimensions (jumbo)

| Feature | Dimension |
|---------|-----------|
| Overall length | 235 mm |
| Ken length | 212 mm |
| Tama diameter | 80 mm |
| Sarado width | 94 mm |
| Big cup diameter | 54 mm |
| Small cup diameter | 51 mm |
| Base cup diameter | 48 mm |

Jumbo-class proportions; tune for printability and the embedded electronics
(IMU + APA102 strip + LiPo) as the CAD develops.
